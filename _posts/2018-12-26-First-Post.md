# SSAFY Start Camp 챗봇 퀘스트
## 1. 구현기능
* 메아리 챗봇
  : 사용자가 보내준 메시지를 그대로 돌려줌

* 로또 챗봇
  : `로또`라는 메시지를 보내면 번호를 추천해준다


* 번역 챗봇
  : `/번역______`이 오면 네이버 API를 통해서 번역된 값을 보내줌

* 유명인 인식(닮은꼴)
  :`사진`이 오면 네이버 API를 통해서 닮은 사람을 말해준다

## 2. 회고
어플리케이션 만들 때 어려운 점
- 챗봇이 어떠한 과정으로 만들어지는지 몰라서 전반적인 흐름을 이해하기 어려웠음
- 오류코드 잡기가 어려웠음

## 3. 보완
보완하고 싶은 점
- 속도가 느린 점을 보완
